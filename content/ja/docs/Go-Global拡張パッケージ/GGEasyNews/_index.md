### 機能
GGEasyNewsは、GGEasyNotifyから起動されるアプリケーションで、GO-Globalのセッション上にパラメタで指定された情報を画面に表示します。利用に当たっては、Admin Consoleで名称GGNewsとしてアプリケーション登録する必要があります。

### 設置方法

1. GO-GlobalサーバでGGEasyNotify.exeを実行してインストールを行います。
2. Admin ConsoleにてGGEasyNewsを登録します。

### 使用方法
1. 設定ファイルGGEasyNews.iniのdefaultセクションで、以下のパラメタを設定して下さい。

| 設定項目 | 説明             |
|----------|------------------|
| message  | 既定のメッセージ （該当するメッセージが無い時に表示） |
| autoClose| メッセージを自動的に閉じる場合の秒数、0 は継続して表示 |
| <メッセージID>  | GGの接続ポートの番号を指定、既定値は491               |
| appName  | 表示するメッセージ、%s (3個まで) は文字列の置換が行われます |


2. 起動オプションは以下の通りになります。

| 設定項目 | 説明             |
|----------|------------------|
| -id　　　| 表示する設定ファイルの <メッセージID> を引数として指定|
| <文字列> | 置換が指定されたメッセージに与える文字列 （3個まで）|

3. 記載方法は以下の通りになります。

```
[default]
message          = 既定のメッセージです。
msg1             = 印刷スプーラーを再起動しました。これにより再ログオンするまでクライアント側のプリンターを選択できません。
msg2             = これはメッセージ２です。
msg3             = サービスを %s より %s 停止します。
;-
; autoClose      = 30
;-
; [使い方の例]
; GGEasyNews.exe -id msg3 "午後5時" "1時間"
;-
```
